# 🏆 Proyecto — Tema 01: Diseño Conceptual Inicial

---

## 🎯 Objetivo

Diseñar el modelo conceptual de una app de cursos online, identificando entidades, atributos y relaciones clave.

---

## Entregable 1 — Entidades y atributos

Define al menos estas entidades:
- `usuarios`
- `cursos`
- `inscripciones`
- `lecciones`

Para cada una, propone:
- clave primaria,
- 4-6 atributos relevantes,
- un atributo obligatorio (`NOT NULL`).

---

## Entregable 2 — Relaciones

Modela las relaciones:
- un usuario puede tener muchas inscripciones,
- un curso puede tener muchas lecciones,
- una inscripción conecta usuario y curso.

Indica cardinalidades (1:1, 1:N, N:M) y cómo resolverías N:M.

---

## Entregable 3 — Reglas de integridad

Especifica 5 reglas mínimas:
1. PK no nulas.
2. Emails únicos.
3. Integridad referencial en inscripciones.
4. Fechas válidas.
5. Estados permitidos (`ACTIVO`, `INACTIVO`).
