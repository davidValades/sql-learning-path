# 🔒 Tema 14: DCL y Seguridad (Data Control Language)

> **"De nada sirve tener la base de datos mejor diseñada del mundo si cualquiera puede entrar y borrar todo."** DCL es el conjunto de sentencias SQL que controla **quién puede hacer qué** dentro de la base de datos. Es la diferencia entre una fortaleza y una casa con la puerta abierta.
