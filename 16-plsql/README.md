## 📋 Descripción

PL/SQL (Procedural Language/SQL) es el lenguaje de programación procedural de Oracle que extiende SQL con estructuras de control, variables, manejo de excepciones y mucho más. Permite crear lógica de negocio directamente en la base de datos.
