## Ejercicio 1 — Bloque anónimo de KPIs

Muestra por `DBMS_OUTPUT` total de pedidos y ticket medio (`AVG(total)`).

<details>
<summary>👉 Ver solución</summary>

```sql
DECLARE
  v_total NUMBER;
  v_ticket NUMBER(12,2);
BEGIN
  SELECT COUNT(*), ROUND(AVG(total),2)
  INTO v_total, v_ticket
  FROM pedidos;

  DBMS_OUTPUT.PUT_LINE('Total pedidos: ' || v_total);
  DBMS_OUTPUT.PUT_LINE('Ticket medio: ' || v_ticket);
END;
/
```

</details>

## Ejercicio 2 — Procedimiento con validación de stock

Crea `sp_reponer_stock(p_id_producto, p_unidades)` que sume stock y falle si `p_unidades <= 0`.

<details>
<summary>👉 Ver solución</summary>

```sql
CREATE OR REPLACE PROCEDURE sp_reponer_stock(
  p_id_producto IN NUMBER,
  p_unidades IN NUMBER
) AS
BEGIN
  IF p_unidades <= 0 THEN
    RAISE_APPLICATION_ERROR(-20010, 'Unidades inválidas');
  END IF;

  UPDATE productos
  SET stock = stock + p_unidades
  WHERE id_producto = p_id_producto;
END;
/
```

</details>

## Ejercicio 3 — Función de clasificación de cliente

Crea función `fn_segmento_cliente(p_id_cliente)` con salida `VIP`, `ACTIVO`, `BASICO` según gasto total.

<details>
<summary>👉 Ver solución</summary>

```sql
CREATE OR REPLACE FUNCTION fn_segmento_cliente(p_id_cliente IN NUMBER)
RETURN VARCHAR2
AS
  v_gasto NUMBER(12,2);
BEGIN
  SELECT NVL(SUM(total),0)
  INTO v_gasto
  FROM pedidos
  WHERE id_cliente = p_id_cliente;

  RETURN CASE
    WHEN v_gasto >= 1000 THEN 'VIP'
    WHEN v_gasto >= 300 THEN 'ACTIVO'
    ELSE 'BASICO'
  END;
END;
/
```

</details>

## Ejercicio 4 — Trigger de auditoría en citas

Crea tabla `auditoria_citas` y trigger `trg_citas_estado` que registre cambios de estado.

<details>
<summary>👉 Ver solución</summary>

```sql
CREATE TABLE auditoria_citas (
  id_aud NUMBER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  id_cita NUMBER,
  estado_anterior CHAR(1),
  estado_nuevo CHAR(1),
  fecha_evento DATE DEFAULT SYSDATE,
  usuario_bd VARCHAR2(100)
);

CREATE OR REPLACE TRIGGER trg_citas_estado
AFTER UPDATE OF estado ON citas
FOR EACH ROW
BEGIN
  INSERT INTO auditoria_citas(id_cita, estado_anterior, estado_nuevo, usuario_bd)
  VALUES (:OLD.id_cita, :OLD.estado, :NEW.estado, USER);
END;
/
```

## Ejercicio 5 — Manejo de excepciones

Escribe un bloque PL/SQL que consulte un producto por ID y maneje `NO_DATA_FOUND`.

<details>
<summary>👉 Ver solución</summary>

```sql
DECLARE
  v_nombre productos.nombre%TYPE;
BEGIN
  SELECT nombre INTO v_nombre
  FROM productos
  WHERE id_producto = 999999;

  DBMS_OUTPUT.PUT_LINE(v_nombre);
EXCEPTION
  WHEN NO_DATA_FOUND THEN
    DBMS_OUTPUT.PUT_LINE('Producto no encontrado');
END;
/
```

</details>
