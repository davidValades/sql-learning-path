## 📝 Descripción

Este repositorio es un diario de aprendizaje y un portafolio de mi camino autodidacta para dominar SQL y las bases de datos relacionales. El objetivo es adquirir conocimientos sólidos de forma progresiva, partiendo desde los conceptos más básicos hasta alcanzar técnicas avanzadas de optimización y casos de uso en el mundo real.

A través de este repositorio iré documentando mi progreso teóricamente y adjuntando los ejercicios prácticos y scripts que vaya desarrollando.

## ⚙️ Metodología

- **Estudio Independiente:** Comprensión profunda de la arquitectura y funcionamiento de las bases de datos.
- **Práctica Continua:** Creación de esquemas, inserción de datos y resolución de consultas progresivamente más complejas.
- **Hoja de Ruta:** El siguiente temario marca el camino a seguir. Los temas se irán marcando con un "check" a medida que los vaya estudiando y dominando.
