# 🔗 Tema 08: Relaciones y JOINs

> 🚧 **Este tema está en construcción.** El contenido se irá completando próximamente.

## 📋 Descripción

Cruzando datos de múltiples tablas. Dominio absoluto de `INNER JOIN`, `LEFT/RIGHT JOIN`, `FULL OUTER JOIN` y `CROSS JOIN`.

## 🔮 Contenido Previsto

- [ ] `INNER JOIN` — Solo registros que coinciden en ambas tablas
- [ ] `LEFT JOIN` — Todos los registros de la tabla izquierda
- [ ] `RIGHT JOIN` — Todos los registros de la tabla derecha
- [ ] `FULL OUTER JOIN` — Todos los registros de ambas tablas
- [ ] `CROSS JOIN` — Producto cartesiano
- [ ] `SELF JOIN` — Unir una tabla consigo misma
- [ ] Sintaxis antigua de Oracle (`(+)`) vs. sintaxis ANSI
