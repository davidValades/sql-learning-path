# 🔗 Recursos del Tema 3.1: CREATE TABLE (Oracle SQL)

Para practicar la creación de tablas y ejecutar consultas con sintaxis nativa de Oracle sin necesidad de instalar bases de datos locales:

1. **[Oracle Live SQL](https://livesql.oracle.com/):** La herramienta oficial de Oracle. Es gratuita (solo requiere crear una cuenta de Oracle), te permite guardar tus scripts y usar la versión más reciente del motor.
2. **[DB Fiddle UK](https://dbfiddle.uk/):** Plataforma web que soporta motores Oracle. Excelente para pruebas rápidas de esquemas y para compartir código.
